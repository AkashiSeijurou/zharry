About the Pack

Auto Updating

Our launcher is auto-updating, that means everytime we make a mod update, the launcher would do everything by itself.

All the best

Mod instability? Maybe, but we get every single feature the mod developers have to offer.
Mod compatability issues? Maybe, but we try to jam the most fun into the smallest package.

Geared towards fun!

Industrial Magic 2 not only includes the best of the mods, you get from other experiances, execpt some of us got bored.
So now, there are more to explore, more to play and more to do.

Online Server Minimap Planned

See a minimap of the server online, updating in real-time, powered by JourneyMap - Unlimited!

Online World Viewer Planned

Because Minecraft Overviewer is a vanilla program, we are not certain if it will work, but if it does, a 3D map of the world will be online!

More to come!

We are still in the progess of making the modpack, but this page will continue to reflect all the changes we make, even though you can't see it! More of those fun mods will be added and some of the unused mods from the last server will be removed.

Official Start Date

It's really up to how fast Harry can get things done, but expect the official opening to be around the end of September to the start of October. Remeber, it takes time to configure our custom server and make the modpack.
